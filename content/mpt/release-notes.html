Apache James MPT

MPT is a scriptable functional test tool particularly suitable for the ASCII protocols beloved by mail standards.

Comments, Questions and Issues

MPT is created by the Apache James Project. Please direct your comments and questions to the relevant list. To report issues, such as bugs, see here.
