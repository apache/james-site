Mailbox Cassandra

This module provides a mailbox implementation for persisting mailboxes (messages, and subscriptions) in a Cassandra cluster.

It supports the Basic and Move capabilities.

Overview

The Cassandra implementation comes with a CassandraMailboxManager and a CassandraMessageManager for persisting users mailboxes.

This implementation also have a CurrentQuotaManager and a MaxQuotaManager, for storing users quotas in a Cassandra cluster.
