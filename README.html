Signatures

Many of the files have been digitally signed using GnuPG. If so, there will be an accompanying file.asc signature file in the same directory as the file (binaries/ or source/). The signing keys can be found in the distribution directory at <http://www.apache.org/dist/james/KEYS>.

Always download the KEYS file directly from the Apache site, never from a mirror site.

Always test available signatures, e.g.,
$ pgpk -a KEYS
$ pgpv james-2.1.tar.gz.asc
or,
$ pgp -ka KEYS
$ pgp james-2.1.tar.gz.asc
or,
$ gpg --import KEYS
$ gpg --verify james-2.1.tar.gz.asc
