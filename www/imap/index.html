Introduction

What Is James IMAP?

A library providing a flexible codec for IMAP, command processors and a sample data access layer. In combination with a socket layer, this library can be used to create an IMAP server.

What Is The Status Of James IMAP?

RFC2060 is functionally complete, tested and usable. However, the data access layer is considered unsatisfactory from design and performance perspectives. Database schema and API changes are expected.

Is IMAP Available In James?

JAMES Server 3 will support IMAP (using this library). A backport to JAMES Server 2.x would require a volunteer.
