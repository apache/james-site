What is jSieve?

jSieve is a Java implementation of the Sieve mail filtering language defined by RFC 3028. jSieve is implemented as a language processor that can be plugged into any internet mail application to add Sieve support.

Sieve Mailets

This library contains mailets which allow Sieve filters to be easily fitting into any enterprise mail server supporting the Mailet API. This library supplies Sieve mail filtering support for James.

For more details, see the Catelog.
