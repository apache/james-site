Downloading mime4j

Go to the Sourceforge download pages and download the most recent release in your preferred format, either mime4j-x.y.tar.gz or mime4j-x.y.zip. Extracting the archived sources will create the directory mime4j-x.y/ which contains the mime4j jar file and the API docs.
