Apache Hupa Demo site.

You can select one of our demo instances:

  • Hupa in Demo mode:
    • Username is set to 'demo' and the password is 'demo'
    • Uses a collection of example messages and stub implementations of IMAP and SMTP servers.
    • In this instance you can test almost the Hupa features, but emails you send will never reach the addressee, they only will be saved in the Sent folder.
  • Hupa for Gmail:
    • It will work with any imap enabled account in gmail.
    • You have to provide a valid gmail username (don't forget the domain part) and your password.
    • Because this instance and gmail servers are far away, and Hupa caching is not enabled, you will notice a latency when using the application.
    • You can read your real messages and use Hupa to send messages to your contacts.
