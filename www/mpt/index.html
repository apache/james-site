What is the Apache James MPT?

Apache James Mail Protocol Tester (MPT) is a framework for the scritable functional testing of ASCII based line protocols. Apache James MPT requires Java 1.5 or higher.

Modules

The main library contains the basic frameworks suitable for creating JUnit tests and for creating standalone tests suitable for independent execution. This contains a lightweight scripting framework suitable for the functional testing of the ASCII line based protocols common in mail. The script contains a list of client sends and templated server responses. The framework runs the client lines against a host system and then compares the server lines received in response against the templates (which may include regex's and so on).

The AntLib module extends the standalone framework to create functional testing tasks suitable for execution against a indepedently running server.
