Standard Mailets

A collection of generally useful reusable mailets and matchers with minimal dependencies.

These mailets form the mail processing backbone of the Apache James Server. They are coupled only to the Mailet API and so are independent of the server.

Releases

Apache Mailet Standard 1.0 is the current release and can be downloaded here.

1.0 is the first independent release of the API. Earlier versions were released as part of the Apache James Server and are versioned accordingly.
