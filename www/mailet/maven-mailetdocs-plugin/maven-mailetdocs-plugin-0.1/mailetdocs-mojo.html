mailetdocs:mailetdocs

Note: This Mojo should be used as a Maven report.

Full name:

org.apache.james:maven-mailetdocs-plugin:0.1:mailetdocs

Description:

Generates catalog and reports on mailets and matchers.

Notes

  • Should only used as a report.
  • Mailets are instantiated during report production.
Attributes:
  • Requires a Maven 2.0 project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Since version: 0.1.

Required Parameters

Name Type Since Description
outputDirectory File 0.1 The output directory.

Parameter Details

outputDirectory: The output directory.
  • Type: java.io.File
  • Since: 0.1
  • Required: Yes
  • Expression: ${project.reporting.outputDirectory}
